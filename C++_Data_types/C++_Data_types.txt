Task 2. Write a C++ console program to execute and print to the screen all the values ​​of mathematical expressions given the values ​​of two integer variables entered by the user. Use +, -, *, /, %.

Task 4. Write a C++ console program to exchange them values ​​of two variables, by bitwise operations.
